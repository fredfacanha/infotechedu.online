DÚVIDAS FREQUENTES

Qual tamanho do pendrive que preciso ter para copiar os arquivos?

Para montar seu Box é recomendável ter um pen-drive de 64GB, de qualquer modelo.

Precisa ativar o Windows e Office?

Sim! Não somos revenda Microsoft. Você terá que ativar o Windows e Office após a formatação. Isso é ensinado no portal do cliente.

O sistema reconhece em GPT e MBR?

SIM. O sistema é baseado em WINPE e reconhece ambas os boots: GPT e MBR.

Essa ferramenta é um MULTIBOOT?

NÃO! Não é uma ferramenta multibook. São ferramentas de automação. Automatiza desde a preparação do pendrive bootável até a formatação e instalação do Windows, já com programas básicos instalados em imagens originais e generalizadas para não ter nenhum tipo de BUG ou Incompatibilidade de Hardware.

O sistema de Formatação define se vai ser melhor instalar Windows 32, ou 64 bits de acordo com a configuração do equipamento?
Não, você deve fazer a escolha da arquitetura manualmente.
Como receberei a ferramenta?

O Box Formate Rápido é um conjunto de arquivos 100% online e composto por 1 tutorial de uso. Você receberá em seu e-mail os dados para acessar a plataforma onde serão entregues todos os conteúdos.

Tenho direito a certificado?

NÃO. Isto é uma ferramenta de implantação e não um curso online.

Posso pagar via PIX?

SIM! Na página de pagamento você poderá escolher pagar via PIX, Boleto ou Cartão de Crédito.

Por quanto tempo vou ter acesso?

Você terá acesso total pelo período de 6 meses. Durante este período você pode acessar o portal quantas vezes desejar e assistir o tutorial e baixar os arquivos.

Os meus dados estão seguros?

Sim. 100% seguros. Todo o processo de compra é gerenciado pela Kiwify, uma das maiores plataformas de produtos digitais do Brasil, através de uma transação criptografada com 128 bits.

Serve para quem NÃO tem experiência com formatação?

SIM! Esta ferramenta foi criada para que qualquer pessoa possa aprender a usá-la de forma didática, simples e prática.
